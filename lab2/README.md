# НКА:
![NFA](assets/nfa.svg)

## Таблица НКА:
| State | $a^2$ | $\varepsilon$ | $ab$ | $b$ | $a$ |      
|-------|-------|-------|-------|-------|-----|
| {0}($\varepsilon$)     | +     | -     | -     | +     | -   | 
| {5}(b)     | -     | +     | -     | -     | -   |
| {6}(ba)     | -     | -     | +     | -     | -   |
| {7}(bb)     | -     | -     | -     | +     | -   |
| {1, 3}(a)     | -     | -     | -     | -     | +   | 



То есть в НКА как минимум 5 состояний.

# Расширенное регулярное выражение:

`^(?!(a|aa|aaaa|aaaaaaa)(?=(aa|b)))a*(aa|b)((bb)+|(aab)+)?$`

Строится исходя из соображения, что 
`(aaa|aaaaa)*` = $a^n$ `\ {a, aa, aaaa, aaaaaaa}`
